Guidance of this website

Getting started:

  • To start to creating a blog post, you must first login
  • With that done. You can now start creating new blog
  • It logs the logged user below and shows the counter of how many blogs you have so far created!

Suggestions

  • Some of the inputs are optional, what is required is ONLY the title!
  • Try and have a default media url and alt, that gets on display if none is provided

Media

  • Media url is meant to be direct link to image, cannot lead to a website containing that image must be direct!
  • To get direct image, on Unplash as example, right click image and "copy image url", something like that. And test it.

Tags

  • Tags will be different for each of us, in this case, I have taken use of checkboxes. You can use open field and use comma as separator.
  • Best solution for typo-mistake occuring, expandable and easier to use.

Body:

  • The way you type content inside the "body" is not necessarily how it will look on fetch and display on your blog post. You will need to look into how to make sure it is taken into consideration the break points in js! Read up here (opens in new tab)
  • If still unsure what I mean, ask chatGPT how to handle breakpoints in textarea.
